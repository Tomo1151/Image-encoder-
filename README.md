# Image-encoder
## Usage
`python encode.py [file name (./img/)] [comp-method] [-debug (optional)]`  
or  
`python encode.py [file name (./img/)] [LZMA (bool)] [LZHAM (bool)] [Deflate (bool)] [-debug (optional)]`  
note : bool => 0 / 1  
  
## Require
- Node.js  
    - pngjs  
- Python (less than 3.9)
    - pylzham
