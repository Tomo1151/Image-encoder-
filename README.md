# Image-encorder
## Usage
`python encode.py [image-path] [comp-method]`  
or  
`python encode.py [image-path] [LZMA (bool)] [LZHAM (bool)] [Deflate (bool)]`  
note : bool => 0 / 1  
  
## Require
- Node.js  
    - pngjs  
- Python (less than 3.9)
    - pylzham  
  
## Note  
if u use this for debug use, plz edit `debug = 0` to `debug = 1` in encode.py
